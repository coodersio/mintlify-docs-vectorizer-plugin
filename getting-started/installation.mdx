## Install the Plugin

Get Vectorized (AI Vectorizer) - Image Tracer installed and ready to use in just a few clicks.

<Steps>
  <Step title="Open Figma Community">
    <Tabs>
      <Tab title="Direct Link (Fastest)">
        Click the button below to install directly from Figma Community:

        <Card
          title="Install Vectorized (AI Vectorizer)"
          icon="download"
          href="https://www.figma.com/community/plugin/1528574912559745892/vectorized-ai-vectorizer-image-tracer"
          color="#16A34A"
        >
          Open in Figma Community ‚Üí
        </Card>

        This will open Figma and take you directly to the plugin page.
      </Tab>

      <Tab title="Manual Search">
        **Option 1: Using Resources Panel**
        1. Open any Figma file
        2. Press `Shift + I` (or click **Resources** in the toolbar)
        3. Click the **Plugins** tab
        4. Search for **"Vectorized (AI Vectorizer) - Image Tracer"** or **"Vectorized"**
        5. Click **Install** next to the plugin

        **Option 2: Using Figma Community**
        1. Go to [figma.com/community](https://figma.com/community)
        2. Search for **"Vectorized (AI Vectorizer) - Image Tracer"** or **"Vectorized"** in the search bar
        3. Click on the plugin from results
        4. Click the **Install** button
      </Tab>
    </Tabs>
  </Step>

  <Step title="Launch the Plugin">
    After installation, run the plugin using any of these methods:

    <AccordionGroup>
      <Accordion title="Method 1: Resources Panel (Recommended)" icon="grid">
        1. Press `Shift + I` or click **Resources** in the toolbar
        2. Go to the **Plugins** tab
        3. Find **Vectorized (AI Vectorizer) - Image Tracer** in your saved plugins
        4. Click to run

        <Tip>
          **Pro tip**: Right-click the plugin and select **"Run"** to launch it without opening the Resources panel each time.
        </Tip>
      </Accordion>

      <Accordion title="Method 2: Quick Actions (Fastest)" icon="bolt">
        1. Press `Cmd + /` (Mac) or `Ctrl + /` (Windows) to open Quick Actions
        2. Type **"Vectorized (AI Vectorizer) - Image Tracer"** 
        3. Press `Enter` when the plugin appears

        This is the fastest way to launch after your first use!
      </Accordion>

      <Accordion title="Method 3: Right-Click Menu" icon="mouse-pointer">
        1. **Right-click** on any image in your canvas
        2. Hover over **Plugins**
        3. Select **Vectorized (AI Vectorizer) - Image Tracer** from the submenu

        Great for when you're already working with an image!
      </Accordion>

      <Accordion title="Method 4: Menu Bar" icon="bars">
        1. Click **Plugins** in the top menu
        2. Select **Vectorized (AI Vectorizer) - Image Tracer** from the dropdown

        Available in both Desktop and Browser versions.
      </Accordion>
    </AccordionGroup>
  </Step>

  <Step title="Verify Installation">
    When the plugin launches successfully, you should see:

    <Check>‚úì Plugin panel opens on the right side of Figma</Check>
    <Check>‚úì **Settings** panel visible on the left with preset options</Check>
    <Check>‚úì **Credit balance** displayed at the top (shows 0 if you haven't purchased yet)</Check>
    <Check>‚úì **Canvas preview area** in the center</Check>
    <Check>‚úì **Convert to SVG** button at the bottom</Check>

    <img src="/images/plugin-interface-labeled.png" alt="Plugin interface overview" />

    <Warning>
      **Not seeing the interface?**

      - Check your internet connection (plugin requires online access)
      - Try refreshing Figma (`Cmd/Ctrl + R`)
      - Make sure you're using a supported Figma version (see requirements below)
      - Restart Figma and try again
    </Warning>

    <Tip>
      **First time user?** You start with **0 credits**. Don't worry‚Äîyou can still explore the interface and test all settings. When you're ready to convert, purchase credits from the plugin. [View pricing ‚Üí](/support/credits-pricing)
    </Tip>
  </Step>
</Steps>

## Troubleshooting Installation

<AccordionGroup>
  <Accordion title="Plugin doesn't appear in my plugins list" icon="circle-xmark">
    **Possible causes:**
    - Plugin wasn't installed successfully
    - You're looking in the wrong Figma account
    - Browser/Desktop cache issue

    **Solutions:**
    1. Try installing again using the [direct link](https://www.figma.com/community/plugin/1528574912559745892/vectorized-ai-vectorizer-image-tracer)
    2. Check if you're signed into the correct Figma account
    3. Clear browser cache (Browser version) or restart Figma (Desktop version)
    4. Check Figma Community to verify installation status
  </Accordion>

  <Accordion title="Plugin panel is blank or won't load" icon="spinner">
    **Possible causes:**
    - No internet connection
    - Firewall/VPN blocking the plugin
    - Outdated Figma version

    **Solutions:**
    1. **Check internet connection** - Plugin requires online access
    2. **Disable VPN/proxy** temporarily to test
    3. **Update Figma** to the latest version
    4. **Restart Figma** completely (quit and reopen)
    5. Try in a **different browser** (if using Browser version)
  </Accordion>

  <Accordion title="'Insufficient permissions' error" icon="lock">
    **What this means:**
    The plugin needs certain permissions to function, but they weren't granted during installation.

    **Solution:**
    1. **Uninstall** the plugin:
       - Go to Resources (`Shift + I`) ‚Üí Plugins
       - Right-click on Vectorized (AI Vectorizer) - Image Tracer
       - Select "Remove plugin"
    2. **Reinstall** using the direct link
    3. **Accept all permissions** when prompted
  </Accordion>

  <Accordion title="Can't find the plugin after installation" icon="magnifying-glass">
    **Try these locations:**

    1. **Resources Panel** (`Shift + I`) ‚Üí Plugins tab
    2. **Quick Actions** (`Cmd/Ctrl + /`) ‚Üí Type "Vec"
    3. **Menu Bar** ‚Üí Plugins ‚Üí Vectorized (AI Vectorizer) - Image Tracer
    4. **Right-click** any image ‚Üí Plugins ‚Üí Vectorized (AI Vectorizer) - Image Tracer

    If still not found, reinstall the plugin.
  </Accordion>

  <Accordion title="Plugin works on Desktop but not Browser (or vice versa)" icon="laptop-mobile">
    **This is normal behavior**

    Plugins installed on Desktop aren't automatically available in Browser, and vice versa. You need to:

    - **Install separately** in each environment
    - Use the same Figma account
    - Credits are shared across both versions
  </Accordion>
</AccordionGroup>
